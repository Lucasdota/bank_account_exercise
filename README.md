# Simulador de Operações Bancárias
Este é um projeto desenvolvido em Java utilizando o **Eclipse IDE**, que simula operações básicas de um sistema bancário. As funcionalidades principais incluem a **abertura de conta bancária, depósito e saque**. Este projeto foi criado como parte do aprendizado e aplicação de conceitos de **Programação Orientada a Objetos (POO)**.

## Funcionalidades
- **Abertura de Conta**: Criação de contas bancárias com dados básicos (titular, número da conta, saldo inicial, etc.).
- **Depósito**: Adicionar fundos à conta.
- **Saque**: Retirar fundos da conta, com verificação de saldo suficiente.

## Conceitos Aplicados
O projeto incorpora diversos princípios fundamentais da programação orientada a objetos:
- **Orientação a Objetos (POO)**: O código foi estruturado em classes e objetos que representam os componentes de um sistema bancário, como contas e operações.
- **Construtores**: Foram implementados construtores para a inicialização de objetos de maneira eficiente, permitindo criar contas com ou sem saldo inicial.
- **Sobrecarga**: A sobrecarga de métodos foi aplicada para possibilitar a criação de contas com diferentes parâmetros (por exemplo, com ou sem saldo inicial).
- **Encapsulamento**: O acesso direto aos atributos das classes é controlado por meio de getters e setters, garantindo que a manipulação de dados seja feita de maneira segura e controlada.

## Como Executar
1. Clone o repositório:
```bash
git clone https://github.com/Lucasdota/bank_account_exercise.git
```
2. Importe o projeto no Eclipse IDE.
3. Compile e execute a classe principal, onde as operações de abertura de conta, depósito e saque podem ser simuladas.

## Tecnologias Utilizadas
- **Java**: Linguagem principal de programação.
- **Eclipse IDE**: Ambiente de desenvolvimento integrado utilizado para o desenvolvimento do projeto.
